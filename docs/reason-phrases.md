# 1行理由テンプレートと語彙

目的: 「特徴×地域×施策」の句型で統一し、説明責任を UI の語彙で確保。

## 句型

```
{特徴} × {地域/区間} × {施策}
```

例:

- `混雑率↑ × 末端区間 × 区間短縮`
- `利用低迷 × ○○地区 × 停留所統廃合`
- `高齢化率↑ × △△丘陵 × DRTゾーン`

## 語彙表（例）

- 特徴: 混雑率↑/↓, 利用低迷, 収支悪化, アクセス不均衡, サービス過剰
- 地域/区間: ○○駅–△△駅, ○○地区, 幹線×支線分岐
- 施策: 減便, 停留所統廃合, 区間短縮, DRTゾーン

## 差分注記

- 類似案の差別化要素を末尾に `[差分: 本数-1]` のように角括弧で付加。
