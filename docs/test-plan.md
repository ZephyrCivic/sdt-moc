# テスト計画（抜粋）

## 画面

- 3カラムレイアウトが崩れない（1280px 以上）
- スケルトン表示で体感2秒以内に 10 カードが目視可能

## GTFS 取り込み

- GTFS ZIP（`gtfs_data/AllLines-20250401_群馬中央バス株式会社.zip`）を選択→地図に路線/停留所が表示
- shapes 無の場合の簡略線 fallback が機能

## ガード条件

- GTFS 未読込時は「シナリオ抽出」が disabled
- 候補 0 件時は「SDTで評価」が disabled

## 状態/同期

- カード選択→地図アノテーションの同期
- 進捗イベントのバッチ適用（フレーム差異なし）
- フィルタ/並び替えでカード=地図が一致

## 非同期/リカバリ

- タイムアウト時に自動再送→失敗継続で手動再送に移行
- 再送で `lastUpdatedAt` が更新される

## KPI/推奨

- 生成直後に KPI 非表示→SDT戻りで表示
- 優先KPI切替で並び/推奨のみが即時変化
