# 候補多様性ルール（空間/施策/強度 × 摂動）

目的: 10案の重複/類似乱立を避け、比較の説得力を担保。

## 軸

1) 空間分散: エリアが過度に重ならない（地物の IoU でしきい値管理）
2) 施策多様: 線/点/面（減便/停留所統廃合/区間短縮/DRTゾーン）を混在
3) 強度バランス: 弱/中/強 を最低各2案以上

## 摂動（微小差分）

- 類似案が閾値を超えて密集する場合、以下の微修正で差別化:
  - 線: 区間の端点を 1–2 駅ずらす / 本数±1
  - 点: 統廃合対象の対を変更 / 距離閾値±10–20m
  - 面: ポリゴンのバッファ±50–100m / ズームレベルに応じた簡略化
- カード上は `[差分: ○○]` を 1行理由末尾に付加。

## 実装チェック

- 生成直後に diversityFlags を立て、ツールチップで根拠を表示。
- 10案いずれも 3軸を合算したスコアが一定以上であること。
