# 非同期進捗/エラー/復旧

目的: SDT 評価が非同期で進む前提で、進捗/失敗/再送の UI を標準化。

## 進捗表示

- カード右上に小UI: `評価 xx%` / `最終更新 hh:mm:ss` / `再送` ボタン
- 進捗イベントは 1フレームに集約してカード=地図へ同時適用

## タイムアウト/部分失敗

- タイムアウト: `errorCode='TIMEOUT'` をカードに表示し、自動再送（1回）→ 失敗継続時は手動再送へ移行
- 部分失敗: 失敗要素を `note` としてカードに残す（説明責任）

## 再送

- 自動: 初回タイムアウト時のみ 1 回
- 手動: ユーザーが `再送` ボタン押下で `POST /mock/sdt/evaluate`（対象IDのみ）
